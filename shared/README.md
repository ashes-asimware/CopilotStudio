# Shared Resources

This directory contains shared resources, utilities, and libraries that can be used across multiple apps and agents.

## Structure

```
shared/
├── README.md              # This file
├── utilities/            # Common utility functions
├── templates/            # Reusable templates
├── configs/              # Shared configuration files
└── docs/                 # Shared documentation
```

## Purpose

The shared directory is intended for:
- Common utility functions and helpers
- Reusable code components
- Shared configuration templates
- Common documentation and guidelines
- Libraries used by multiple apps or agents

## Guidelines

- Only include truly shared resources that benefit multiple apps/agents
- Document all shared resources clearly
- Maintain backwards compatibility when updating shared resources
- Version shared components when necessary
- Keep dependencies minimal and well-documented
