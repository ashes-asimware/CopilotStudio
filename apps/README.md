# Apps

This directory contains Copilot Studio generated applications.

## Structure

Each application should be organized in its own subdirectory with the following recommended structure:

```
apps/
├── app-name/
│   ├── README.md          # Application documentation
│   ├── src/              # Source code
│   ├── config/           # Configuration files
│   ├── assets/           # Static assets
│   └── docs/             # Additional documentation
```

## Adding a New App

1. Create a new directory with a descriptive name for your application
2. Include a README.md with:
   - Application description
   - Setup instructions
   - Usage guidelines
   - Dependencies
3. Organize your code and resources in appropriate subdirectories
4. Document any specific requirements or configurations

## Guidelines

- Use clear, descriptive names for application directories
- Keep each application self-contained
- Include all necessary documentation
- Follow consistent naming conventions
